一般正常的启动文件主要分成三部分

  • [Unit] 段: 启动顺序与依赖关系

  • [Service] 段: 启动行为,如何启动，启动类型

  • [Install] 段: 定义如何安装这个配置文件，即怎样做到开机启动

可以看出，/etc/rc.local 的启动顺序是在网络后面，一般默认是没有 Install 段，也就没有定义如何做到开机启动，所以显然这样配置是无效的。 因此我们就需要在后面帮他加上 [Install] 段:
