# chrome深色模式

## Android

```
chrome://flags/#darken-websites-checkbox-in-themes-setting
```

## PC

```
chrome://flags/#enable-force-dark
```

# chrome 多线程下载
默认关闭

```
chrome://flags/#enable-parallel-downloading
```

# 解除赋予权限去除“由贵单位管理”功能选项

谷歌 Chrome 浏览器地址栏输入 `Chrome://policy` ，即可看到调用企业策略的第三方软件或者是用户主动安装的第三方扩展等

1. Windows系统按WIN+R输入`RegEdit`运行注册表管理
2. 删除注册表`计算机\HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Google\Chrome`下对应的文件夹（可以直接删除整个`Chrome`目录）

# chrome外部参数设置

>在“快捷方式”选项卡的“目标”字段末尾添加参数，添加多个参数需要用空格隔开

## 改善深色主题下打开 Chrome 时会出现短暂的白屏闪烁

>最新版已修复

```
--enable-features=RemoveRedirectionBitmap
```

## 忽略网站SSL证书错误

```
--ignore-certificate-errors --test-type
```
