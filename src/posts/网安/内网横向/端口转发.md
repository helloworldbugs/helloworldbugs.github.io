# linux

利用nc进行本地端口转发，本地8888端口转发到22端口

```
nc -lp 8888 -c "nc 127.0.0.1 22"
```


# window

## netsh

 - netsh interface portproxy add v4tov4 listenaddress=localaddress
 - listenport=监听的端口
 - connectaddress=被攻击者ip服务器
 - connectport=被攻击者服务端口
 - protocol=tcp

查看转发规则

```
netsh interface portproxy show all
```

侦听本地端口5555转发到指定ip的端口6666！

```
netsh interface portproxy add v4tov4 listenport=5555 connectaddress=10.1.1.110 connectport=6666
```

删除指定的端口转发规则

```
netsh interface portproxy delete v4tov4 listenport=5555 listenaddress=10.1.1.110 connectport=6666
```

删除所有规则

```
netsh interface portproxy reset```
